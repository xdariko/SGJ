# The Last Technician

The Last Technician — игра, созданная для **Siberian Game Jam Май 2026**.

После столкновения с метеоритом грузовая “Аврора-9” превращается в летающий кошмар из слизи, костей и плохих кадровых решений. Экипаж заражён, в коридорах монстры, а тело главного героя начинает выдавать новые способности без согласия пользователя.
Используй всё, что отрастёт, выстрелит или хрустнет, чтобы пережить эту смену и добраться до выхода.
